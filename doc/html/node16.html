15.4 Sharing data with other applications

You may extract subscribers, owners and editors for a list from any of :

  • a text file

  • a Relational database

  • a LDAP directory

See user_data_source list parameter 20.2.1, page .

The three tables can have more fields than the one used by Sympa, by defining these additional fields, they will be available from within Sympa's authorization scenarios and templates (see 7.10.11, page  and 7.10.12, page ).

See data inclusion file 17.7, page .


15.5 Subscriber count

The number of subscribers of a list can be get from an external application by requesting function 'subscriber_count' on the Web interface.

Example: http://my.server/wws/subscriber_count/mylist
